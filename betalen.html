Veilig betalen

Kies je pakket en activeer je listing vandaag nog.

Je betaalt veilig via Stripe Checkout in testmodus. Na betaling sturen we je terug naar de bedanktpagina en kan je listing worden ingepland voor activatie.

Wat je vandaag regelt

  • Directe selectie van Start, Plus of Full Service.
  • Veilige Stripe hosted checkout zonder verborgen kosten.
  • Activatieflow richting /bedankt.html?betaald=true na afronding.

Selecteer je VendaCuraçao pakket

Klik op het pakket dat bij je verkoop past. Via de URL-parameter ?pakket=start|plus|full wordt je keuze automatisch voorgeselecteerd.

Technische notitie

Deze pagina gebruikt Stripe Checkout met een placeholder publishable key. Voor productie heb je een server endpoint nodig dat een Checkout Session aanmaakt en daarna een redirect geeft naar /bedankt.html?betaald=true. Voeg later ook een webhook endpoint toe voor betalingsbevestiging en statusupdates in je backend.
